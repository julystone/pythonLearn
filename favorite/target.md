# 概述

## 概述

本文档用于说明菜单功能需求规格，菜单功能及排列顺序如下图所示：

![](data:image/png;base64...)![](data:image/png;base64...)

当手机长度不够右菜单需要上下滑动时，顶部未登录、已登录账号位置固定不跟随滑动。

## 释意

在本文档内，除文意另有所指，下述词语具有以下含义：

### 配置保存类型

**不做保存** 配置不做保存，离开页面即重置

**暂时保存** 配置保存内存，直至本次app关闭一直保存

**长期保存** 配置保存本地，直至app卸载重装一直保存

**云端保存** 配置存储云端

### 价格键盘类型

**简易价格键盘** 只显示最小变动价、涨停、跌停；显示数字、小数点、正负号、增减按钮和回删按钮；点击价格键盘右上角【收起】按钮和键盘以外的区域，收起键盘。

**价格键盘** 显示最小变动，涨停，跌停；支持快速切换市价、对手价、最新价、排队价、超价；支持增加FOK、FAK指令，指令可不选，可单选；显示数字、小数点、正负号、增减按钮和回删按钮；点击价格键盘右上角【收起】按钮和键盘以外的区域，收起键盘；点击超价，弹toast提示：以XX价为基准价，超价点位：N。

**手数键盘** 显示可开仓：多、空手数，可平仓手数或无持仓，每手保证金；显示数字、增减按钮和回删按钮；点击手数键盘右上角【收起】按钮和键盘以外的区域，收起键盘。

# 交易登录

## 界面唤起

在**自选行情界面**、**行情报价界面**、**资讯界面**点击底部【交易】按钮，交易账号未登录时，进入**交易登录界面**（仅安卓）。

在**自选行情界面**、**行情报价界面**，长按合约点击弹框【交易】按钮，交易账号未登录时，进入**交易登录界面**。

在**行情图表界面**点击顶部【闪电下单】按钮、【画线下单】按钮、底部【交易】按钮（仅安卓），交易账号未登录时，进入**交易登录界面**。

交易账号未登录时，点击菜单中【交易登录】、【云条件单】、【止损止盈】、【止损开仓】、【交易相关】按钮，进入**交易登录界面**。

点击**交易界面**账号选择功能中【添加账号】按钮，进入**交易登录界面**。

账号登录后，显示当前选中交易的账号及已有n个账号登录，n为当前连接正常的账号数。

已登录一个及以上交易账号时，点击菜单中【多账号登录】按钮，进入**交易登录界面**。

登录完成后，跳转到待响应的界面，如打开闪电下单功能、进入云条件单界面等。

## 开户公司选择

用于选择开户公司，支持国内、外盘，实盘、模拟，分类显示，默认显示国内期货实盘列表。

### 功能唤起

点击选择框，进入**选择开户公司界面**。

### 界面设计

如下图所示：

![](data:image/png;base64...)

显示区分国内期货和外盘期货、实盘和模拟，默认显示国内实盘。

### 交互操作

1、 点击开户公司选择框，进入**选择开户公司界面**。

2、 默认显示国内实盘，用户有记住账号后，默认跳转到上一次登录账号所属的板块，如外盘实盘，不定位到具体公司。

3、 点击某公司名称后，携名称返回**交易登录界面**。点击返回按钮，返回**交易登录界面**，不修改当前输入框内状态。

4、 通用包第一次打开登录界面，开户公司输入框默认为空，单独包可配置默认选中公司。

### 关联设置

无。

## 交易接口选择

### 功能唤起

无。

### 界面设计

如下图所示：

![](data:image/png;base64...)

根据云端文件排列顺序显示交易接口。？？

### 交互操作

1、 点击选择框，弹出该开户公司下属关联交易接口的下拉选择框；若当前未选中开户公司，下拉框为空，弹toast提示：请先选择开户公司。

2、 交易接口默认显示当前开户公司的第一个接口；若本地有登录信息则显示最后一次登录的接口；下拉选择框按顺序显示，当前选中的接口标红显示在第一行。

3、 点击时选中接口字体显示红色，原选中接口文字显示黑色；点击接口与原选中接口为同一个时，该接口文字显示红色。

4、 接口获得焦点且完成点击事件，将该接口填入输入框并收起下拉弹框。

5、 接口失去焦点未完成点击事件，该接口字体显示黑色，原选中接口文字显示红色（显示不下则不显示），下拉弹框不收起，等待用户重新选择。

6、 再次点击交易接口选择框或点击下拉选择框外区域，收起下拉弹框。

7、 下拉框最多显示四个，不满四个则按已有数量进行显示；若选中接口的其他待选接口数量大于3且选中接口为倒数前三，则下拉框显示最后四个接口。

### 关联设置

无。

## 交易账号输入

### 功能唤起

无。

### 界面设计

如下图所示：

![](data:image/png;base64...)

若用户已有登录记录且保存了账号，交易账号输入框默认显示最后一次登录的账号信息。

交易账号输入框默认显示下拉框按钮，获得焦点时，右侧显示【⮾】，失去焦点则隐藏【⮾】。

下拉框中显示已经记住账号、密码且登录过的交易账号，最近一次登录的账号显示在最上面。

### 交互操作

1、 点击交易账号输入框，弹出系统键盘，输入框右侧显示【⮾】按钮。

2、 点击【⮾】按钮，清空输入框内全部字符。

3、 点击右侧【下拉】按钮，如本地未保存账号，则点击后无响应；如本地保存有一个及以上的交易账号，则弹出选择账号下拉框，下拉框最多显示四个，保存账号大于四个则滚动条展示，按最后一次登录的顺序进行排序，最后一次登录的账号置顶显示。

4、 有本地保存的交易账号，账号输入框默认显示最近登录一次的账号；当前有已经登录的账号，账号输入框默认显示最后一个登录的账号；下拉框标红当前已经登录的账号。

5、 点击下拉弹框中的账号，将选中账号保存在本地的信息回填相关输入框。

6、 点击账号右侧的【⮾】按钮，弹出确认框：不再保存该账号，点击【确定】按钮，关闭弹框并删除该账号相关信息；点击【取消】按钮，关闭弹框不进行删除操作。

7、 当交易接口选择北斗星系统，用户输入的账号中含有英文字母，自动显示成大写字母。

### 关联设置

无。

## 记住账号密码

### 功能唤起

无。

### 界面设计

默认勾选保存账号，不勾选保存密码。

北斗星后台，隐藏保存账号、保存密码的选项。

报价涨跌、K线查价框涨跌点击后均进入右图设置界面，设置选项一致，均为开关样式，证券市场昨结选项均置灰，选中后按钮显示为红底白字。

### 设置说明

1、 涨跌计算区分报价和K线查价框，报价和K线查价框均支持针对内盘期货、外盘期货、黄金现货和证券市场单独设置涨跌计算基准价，其中内盘期货、外盘期货、黄金现货支持昨收、昨结、今开价格作为涨跌计算基准价，证券市场仅支持昨收、今开。

2、 涨跌=最新价-基准价。

3、 涨幅=（最新价-基准价）/ |基准价|。

4、 基准价=0，涨幅=0。

4、 设置完立即生效且设置长期保存。

**报价抬头计算公式：**

沉淀资金 = 双边持仓量\*最新价\*交易单位\*保证金比例

资金流向 = (双边持仓量\*最新价–(双边持仓量-日增仓)\*昨收)\* 交易单位\* 保证金比例

*(注：国内期货合约上述公式中持仓量按双边计算，需对持仓量乘以2)*

期货速涨 =（最新价 - 前1分钟最新价）/前1分钟最新价

股票速涨 =（最新价 - 前5分钟最新价）/前5分钟最新价

现涨 = 最新价–前一笔最新价

投机度 = 成交量/持仓量

趋势度 =（最新价 - 开盘价）/（最高价 - 最低价 ）

流入比例 = 资金流向/沉淀资金

期货振幅% =（最高价 - 最低价）/ 昨结算\*100

股票振幅% =（最高价 - 最低价）/ 昨收盘\*100

结涨 = 结算价-昨结算

日增仓 = 今日持仓量-昨日持仓量

日增仓比% = 今日增仓/昨日持仓量

量比 = 今日每分钟平均成交量/前5日每分钟平均成交量

成交金额 = 当天累计成交额 = 均价\*成交量\*合约单位

活跃度 = 每分钟现金流量的指数加权平均，现金流量=1分钟内的成交金额\*保证金比例

委比% = (委买手数-委卖手数)/(委买手数+委卖手数)

日强弱度 = 日股票收益率/日指数收益率\*100

其中：日股票收益率 = 当天价格/前一天价格

日指数收益率 = 当天指数价/前一天指数价

周强弱度 = 最近5日强弱度累加和/5

月强弱度 = 最近20日强弱度累加和/20

换手% = 换手率=成交量/流通总股数（总股本）

市净率 = 现价/每股净资产

市盈率 = 现价/每股盈利

胜率：（盈利次数＋持平次数） / 交易次数

盈利率：总平仓盈亏 / 期初权益

最大回撤：权益达到高点回撤幅度最大的一笔

最大盈利：统计期间平仓盈利最大的一笔

最大亏损：统计期间平仓亏损最大的一笔

最大收益率：统计期间收益率的最大值

最大风险度：单日资金使用率的最大值

每笔平均盈亏比：每笔平均盈利 / （-每笔平均亏损）

每笔平均盈利（盈利单）：总盈利 / 总盈利笔数

每笔平均亏损（亏损单）：总亏损 / 总亏损笔数

每笔平均净盈亏：每笔平均盈利+每笔平均亏损

最长持仓时间：持仓保留最长的时间

最短持仓时间：持仓保留最短的时间

平均持仓时间：持仓保留平均时间

收益率：总盈亏 / 当日（期初权益+出入金）

每日盈亏：当日期未权益-当日期初权益-当日出入金

最大本金收益率：累计盈亏 / 最大投入本金

累计收益率：累计盈亏 / （起始日期的期初权益+期间总出入金）

累计盈亏：当日期末-起始日期初-截止当日累计出入金

最大投入本金：统计期间当日本金的最大值

当日本金：（上日权益-截止上日累计利润）+当日入金

账户累计收益率：总盈利 / 期初权益

交易收益率：账户累计收益率 / 账户资金使用率

当日净值：（当日权益+当日出金） / （上日权益+当日入金）

净值增长率：上日累计净值\*当日净值

品种盈亏：按照交易手数排序前三品种的盈亏
